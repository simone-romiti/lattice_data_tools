This folder contains modules needed to deal with color matrices in lattice gauge theories.

- `suN`: generic routines for dealing with SU(N) matrices
- `configuration.py`: lattice configuration of color matrices, SU(N) links, SU(N) locally gauge-transforming objects
- `parallel_transport.py`: parallel transporters (products of links along a line)
- `loops.py`: loops on the lattice
